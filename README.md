# VapVuup

Projeto web desenvolvido com HTML, CSS e JavaScript, hospedado através do GitHub Pages.

## 🚀 Demonstração

Acesse o projeto online:

**https://danielpscosta.github.io/vapvuup/**

## 📋 Sobre o Projeto

O VapVuup foi desenvolvido com o objetivo de oferecer uma interface moderna, responsiva e intuitiva para os usuários, proporcionando uma boa experiência de navegação em dispositivos desktop e mobile.

## ✨ Funcionalidades

* Layout responsivo
* Interface moderna e intuitiva
* Navegação simples e rápida
* Compatível com dispositivos móveis
* Hospedagem via GitHub Pages

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub Pages

## 📂 Estrutura do Projeto

```text
vapvuup/
├── index.html
├── css/
├── js/
├── assets/
└── README.md
```

## ⚙️ Como Executar Localmente

1. Clone o repositório:

```bash
git clone https://github.com/danielpscosta/vapvuup.git
```

2. Acesse a pasta do projeto:

```bash
cd vapvuup
```

3. Abra o arquivo `index.html` em seu navegador.

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em diferentes tamanhos de tela, incluindo:

* Desktop
* Tablet
* Smartphone
